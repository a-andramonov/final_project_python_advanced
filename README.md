# final_project_python_advanced
Бот в Телеграм для игры в крестики нолики.
Сначала надо запустить программу в питоне
Заходите в телеграм бота, выбираете /start
В появляющемся окне выбираете место для хода(вы играете за крестики)
Бот в случайном порядке выбирает куда поставить нолик
Результат игры как в классической игре, нужно собрать линию из трех подряд идущих крестиков для победы
Если бот первее собрал линию из ноликов, то выйиграл он
